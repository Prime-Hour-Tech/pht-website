# pht-website

A static-generated marketing site built with Astro, with content managed in Sanity, deployed to Vercel.

## Status

## Stack

- **Frontend:** [Astro](https://astro.build/) (static output)
- **CMS:** [Sanity](https://www.sanity.io/) (headless, schemas-in-code)
- **Hosting:** [Vercel](https://vercel.com/) (edge CDN)
- **Styling:** Tailwind CSS
- **Package manager:** pnpm (Node 20 LTS)

## Prerequisites

- [Node.js 20 LTS](https://nodejs.org/) — the included `.nvmrc` pins the version for nvm / nvm-windows users.
- [pnpm](https://pnpm.io/installation).
